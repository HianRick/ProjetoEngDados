# Tabelas Geradas

As tabelas produzidas pelo script Faker são armazenadas em:

```
/tabelas/
```

Arquivos exportados:

- `tabela_vendas_vinhos.csv`
- `tabela_vendas_vinhos_2.csv`
- `tabela_vendas_vinhos_3.csv`

Cada arquivo representa um lote de dados gerados.

---

## Estrutura dos dados

| Campo | Descrição |
|---|---|
| cliente | Nome do comprador |
| cidade | Cidade de origem |
| vinho | Tipo de vinho |
| quantidade | Quantidade vendida |
| valor | Valor monetário da venda |
| data_venda | Data da transação |

Esses arquivos são consumidos nos notebooks para análise posterior.
